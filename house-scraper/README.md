# Scraper project

## Technologies used

- Windows O.S.
- Python (3.9.10 requiered)
- Selenium 4.1.3
- Requests 2.27.1
- Beautiful Soup 0.0.1
- Chrome web driver

## How to launch

1. Open a terminal and navigate to this folder
2. Execute `setup-dev.bat` (`.\setup-dev.bat`)
3. Execute the program: `.\.venv\Scripts\python.exe scraper/__main__.py`
